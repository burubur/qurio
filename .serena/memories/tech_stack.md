# Tech Stack

**Core Services:**
- **Orchestration/API:** Go 1.22+
- **Frontend:** Vue 3 + TypeScript
- **Document Processing/OCR:** Python 3.10+ (Docling Library)
- **Vector Database:** Weaviate OSS (v1.24+)
- **Metadata Database:** PostgreSQL (v15+)
- **Task Queue:** NSQ

**AI & ML:**
- **Embeddings:** Google Gemini Embedding API (`gemini-embedding-001`)
- **Reranking (Optional):** Jina AI v2, Cohere Rerank v3
- **Protocol:** Model Context Protocol (MCP) JSON-RPC 2.0

**Infrastructure:**
- **Containerization:** Docker, Docker Compose
- **OS:** Linux, macOS, Windows WSL2
